EGX Rights-Issue Performance Tracker

Every Egyptian Exchange subscription-right that traded as a separate instrument since April 2021, shown one by one with how it performed over the window it was tradable.

DATA NOTE instruments carry live EGX trading stats — the full-life price range, traded volume/value and number of trades pulled from EGX's Historical Statistics → Stocks Data tool (egx.com.eg) for the rights instruments it currently exposes. EGX publishes these as a performance envelope + a chart image, not a downloadable per-day OHLCV series, so each right shows its real trading range and liquidity rather than a fabricated daily line. The remaining instruments (2021–2025) are carried from the scraper's verified master list with their windows and capital-increase economics; their day-by-day series is not exposed by any public source (it predates EGX's 3-year tool window).
